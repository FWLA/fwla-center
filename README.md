Central server for FWLA-Center system.

## Build

```shell
./mvnw clean install
docker build -t ihrigb/fwla-center:latest -t ihrigb/fwla-center:<version> .
docker push ihrigb/fwla-center:latest
docker push ihrigb/fwla-center:<version>
```
